This project analyzes Olympic Games data to identify patterns in:

- Gender participation

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

Dataset
The dataset contains historical Olympic data including:
- Athlete Name
- Country
- Year
- Sport
- Event
- Medal

(Source: Kaggle Olympics Dataset)

Key Analysis
- Top medal-winning countries
- Medal trends over years
- Male vs Female participation
- Most successful athletes
